DNS Lookup

Resolve a domain to its IP address using an secure HTTPS connection. This tool sends requests from this device and your IP address.

Notes

  • TTL is a Time to Live, the duration for caching a DNS record in seconds.
  • The SOA (Start of Authority) record contains: the primary name server, the hostmaster's email (first dot . instead of @), the serial number, and the refresh, retry, expire, and minimum TTL values, all measured in seconds.
  • Some domains, usually belonging to large companies, can have multiple IP addresses for different regions. For example, a user in Japan may receive other results than a user in Africa.
  • There are domains without type A records; they do not serve IPv4 requests, only IPv6. However, more domains do not have type AAAA records (IPv6).
  • See also the IP range calculator.
