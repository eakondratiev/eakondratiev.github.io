Detect a File Type

This online application inspects a file signature and classifies it according its content. The file description, its size and MIME type will be shown. All files processed by your computer/device. More than one hundred file types supported. If several files selected, only the first one is processed. To verify your file's integrity, use the online tool to calculate the file's SHA-256 checksum.
